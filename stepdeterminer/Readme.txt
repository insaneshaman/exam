Программа определяет результат сложения функции, которая представляет
собой последовательность ряда: 
F = (x+2)/x-(x+4)/2x+...+(x+2n)/x*n
Функция убывающая, пользователь вводит значение х и значение шага,
на котором функция прерывается и выводит значение функции и количество
шагов, совершённое для достижения заданного шага
------------------------------------------------------------------------
Program determines the result of addition of function steps, that 
is sequence of number's row:
F = (x+2)/x-(x+4)/2x+...+(x+2n)/x*n
Function is decreasing and user enters x and needed step. Then 
function adds while the step is more than needed step, entered 
by user. Then program shows function value and number of steps
that were gone
