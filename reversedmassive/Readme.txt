Программа принимает на вход данные о массиве (кол-во строк и столбцов),
пользователь может выбрать способ задания элементов массива (вручную 
или сгенерировав случайные числа с указанием диапазона этих чисел),
а затем программа выводит этот массив и его же, но перевёрнутым 
на 180 градусов.
----------------------------------------------------------------------
Program recieves array capacity (number of rows and columns), user can
choose the way of array elements assignment (manually or with random
number generation) and then program shows the array and flipped 180 degrees
array.
